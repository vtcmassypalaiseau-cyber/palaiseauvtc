# Palaiseau VTC

Site de réservation VTC pour Massy–Palaiseau et l’Île-de-France.

## Description

Ce dépôt contient une page HTML statique (`index.html`) qui présente le service Palaiseau VTC et permet de mettre en avant :
- les trajets VTC en Île-de-France (gares, aéroports, déplacements pro, famille, etc.) ;
- une mise en page modernisée adaptée au mobile ;
- un déploiement automatique via Netlify à partir de ce dépôt GitHub.

## Déploiement

Le site est déployé automatiquement sur Netlify à chaque push sur la branche `main`.

Étapes générales :
1. Modifier `index.html` en local.
2. Commiter et pousser les changements sur GitHub.
3. Netlify reconstruit et met à jour le site quelques secondes plus tard.

## Contact

Pour toute modification ou évolution du site, utilisez ce dépôt ou créez une nouvelle branche.
